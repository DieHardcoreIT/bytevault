How It Works

ByteVault is a project that allows you to securely store and reconstruct files using a unique key-based system. Upload a file to generate a key file, and use the key file along with the server data to reconstruct the original file.

Important Notes

Validity: Key files are valid for a maximum of 7 days. After that, the server data required for reconstruction will no longer be available.

Privacy Notice: All processing happens locally in your browser. No files or parts of files are uploaded to or stored on the server at any time.
